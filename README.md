# ארנונה חכמה (Arnona Chacham)

בדיקת ארנונה חינמית לתל אביב — מזהה אזור שגוי, חיוב‑יתר והנחות שלא נוצלו, ומפיק תיק השגה מוכן להגשה עצמית. גרסת BETA.

חלק ממשפחת "הבדיקות הפיננסיות החינמיות" עם [מס חכם](https://maschacham.com).

## מה זה עושה
- **מנוע תעריפים** — צו הארנונה הרשמי של עיריית תל אביב‑יפו 2026 (מגורים: 6 סוגי בניין × 5 אזורים; לא‑מגורים: 3 אזורים).
- **זיהוי אזור אוטומטי מכתובת** — geocode (OSM) → point‑in‑polygon מול ה‑GIS של עיריית ת"א (ArcGIS layer 589) → השוואה לאזור שבשובר (הטעות הנפוצה ביותר).
- **מאתר הנחות** — לפי פרופיל המשתמש (אזרח ותיק, נכות, הורה יחיד, עולה, נכס ריק).
- **תיק השגה (PDF)** — עילות + נוסח השגה מוכן + צעדי הגשה (90 יום, מודד, ועדת ערר).

## ארכיטקטורה
עמוד יחיד סטטי (`index.html`), vanilla JS + Tailwind. כל החישוב **דטרמיניסטי בקוד**; אין שרת, אין שמירת נתונים (הכל בדפדפן).

## הרצה מקומית
```
npx serve . -l 8090
```

## דיפלוי
Cloudflare Workers (assets סטטיים) — `wrangler deploy`. ראה `wrangler.toml`.

## מגבלות / סטטוס
- **תל אביב בלבד** (זיהוי האזור מבוסס ה‑GIS הייחודי של ת"א). שאר המוצר עובד בכל עיר עם הזנת אזור ידנית.
- ה‑geocoder הוא OSM (פרוטוטייפ) — גרסה אמיתית תשתמש ב‑geocoder ישראלי רשמי.
- פער שטח מסומן, לא מוכח (דורש מודד מוסמך).
- **שירות מידע/עזרה‑עצמית בלבד — אינו ייעוץ משפטי.** הסכום הסופי וההחזר נקבעים ע"י העירייה / ועדת ערר.
